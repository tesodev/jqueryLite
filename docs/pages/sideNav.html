.sideNav()

A side navigation menu can be created by using following codes.

<div class="side-nav" tabindex="0" style="display: none">
    ...
</div>

You must initialize sideNav component.

    $(buttonToOpenSideNav).sideNav({
        menuWidth: 250,
        edge: 'left', // left or right. default left
        target: sideNavSelector,
        closeOnClick: false, //click on sidenav default false
        time: 0.5, // animation time. default 0
        onOpen: function (el) {
            console.log("sidenav opened, ", el)
        },
        onClose: function (el) {
            console.log("sidenav closed, ", el)
        }
    });

methods:

    $(buttonToOpenSideNav).sideNav("destroy");
    $(buttonToOpenSideNav).sideNav("open");
    $(buttonToOpenSideNav).sideNav("close");
    $(buttonToOpenSideNav).sideNav("toggle");
